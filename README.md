# Minibob
Simple blog application using laravel 
Requirements 

1. php 8+
2. xampp
3. composer 2
4. mysql


Setting up 1 :
| Go to your htdocs folder in terminal type:

$ git clone https://github.com/joyal777/Minibob.git

$ cd Minibob

$ composer install or update

$ php artisan key:generate

$ php artisan migrate

$ php artisan serve

............................................
